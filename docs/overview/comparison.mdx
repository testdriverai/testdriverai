### Application support

TestDriver operates a full desktop environment, so it can run any application.

| Application       | TestDriver | Playwright | Selenium |
| :---------------- | :--------- | :--------- | :------- |
| Web Apps          | ✅         | ✅         | ✅       |
| Desktop Apps      | ✅         |            |          |
| Chrome Extensions | ✅         |            |          |
